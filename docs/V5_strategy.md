# V5 缩量回踩反弹策略 — 最优参数

## 核心逻辑
**不追涨，买在起涨点。** 找强势股（RPS≥70）在缩量洗盘后首次放量反弹的买入时机。

## 入场条件（全部满足）

| 条件 | 参数 | 说明 |
|------|------|------|
| RPS | ≥ 70 | 全市场排名前30%的强势股 |
| 今日涨幅 | -1% ~ 3% | 微涨或小幅反弹，不追大涨 |
| 今日量比 | ≥ 1.2 | 放量确认 |
| 前日涨幅 | -2% ~ 1% | 前一日回调或微涨，不是连续上涨 |
| 近3日最低量比 | ≤ 0.7 | 缩量洗盘确认 |
| 均线排列 | MA5 > MA10 > MA20 | 多头趋势 |
| 股价位置 | close > MA10 | 站上10日线 |
| 均线支撑 | 距MA10≤5% 或 距MA20≤6% | 在支撑位附近 |
| 换手率 | > 1.5% | 有流动性 |
| 价格 | > 5元 | 排除低价股 |
| 排除 | ST、688/92/8/4开头 | 排除风险股 |

## 回测结果

**区间**: 2025-01-02 ~ 2026-04-30 (320个交易日)

| 持有期 | 胜率 | 平均收益 | 样本数 |
|--------|------|----------|--------|

**VS 原V4组合策略（追涨逻辑）**：

| 持有期 | V5胜率 | V4胜率 | 提升 |
|--------|--------|--------|------|
| 1天 | 50.6% | 38.2% | +12.4% |

## 策略要点

1. **前日回调是关键** — 前日跌-2%~涨1% 保证了不是追连续上涨的股票
2. **缩量洗盘是核心** — 近3日有量比≤0.7说明主力在洗盘
3. **今日放量是信号** — 量比≥1.2 说明洗盘结束，资金开始回流
4. **RPS≥70 是底线** — 只做强势股，不做垃圾股反弹
5. **3天持有最优** — 胜率63.3%，收益+0.90%；5天后收益转负，说明反弹周期短

## 风险提示

- 样本量79笔（320个交易日），约每4天1只，信号不频繁
- 月度胜率波动：好的月份70%+，差的月份可能40%以下
- 牛市/震荡市效果好，单边下跌市可能失效
- 建议配合大盘状态过滤（大盘下跌时停用）
