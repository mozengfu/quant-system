# 需求定义

## v1 需求

### 策略打磨与回测

| ID | 需求 | 说明 |
|----|------|------|
| REQ-01 | V4 组合策略持续验证 | 定期回测、收益/风险指标追踪 |
| REQ-02 | 策略参数优化工具 | 参数扫描、自动寻优 |
| REQ-03 | 多策略对比回测 | 同条件对比不同策略表现 |

### 模拟交易自动化

| ID | 需求 | 说明 |
|----|------|------|
| REQ-04 | 自动选股扫描 | 每日定时扫描候选股票，减少人工干预 |
| REQ-05 | 自动下单执行 | 根据信号自动执行买入/卖出 |
| REQ-06 | 持仓自动调整 | 动态调仓、再平衡 |

### 风控体系

| ID | 需求 | 说明 |
|----|------|------|
| REQ-07 | 自动化止损止盈 | 盘中 5 分钟轮巡查，自动触发 |
| REQ-08 | 仓位管理 | 市场状态自适应仓位控制 |
| REQ-09 | 市场状态判断 | 趋势/震荡/恐慌/过热自动识别 |

### ML 模型迭代

| ID | 需求 | 说明 |
|----|------|------|
| REQ-10 | ML 训练流水线规范化 | 统一训练入口、版本管理 |
| REQ-11 | 模型性能监控 | 预测准确性跟踪、漂移检测 |
| REQ-12 | 模型清理 | 废弃模型清理，当前仅保留最新版本 |

### 监控与报表

| ID | 需求 | 说明 |
|----|------|------|
| REQ-13 | 交易记录可视化 | 收益曲线、胜率、盈亏比 |
| REQ-14 | 绩效看板 | 资金曲线、夏普、最大回撤等核心指标 |

## v2 需求（延期）

| ID | 需求 | 说明 |
|----|------|------|
| REQ-15 | 多用户支持 | 注册/权限/独立持仓 |
| REQ-16 | 实盘接口接入 | 券商 API 对接自动下单 |
| REQ-17 | 社区功能 | 策略分享、跟单 |

## 外延（不纳入）

- 移动端 App（保持 Web 端优先）
- 海外市场（专注 A 股）
- 高频交易（A 股 T+1 规则限制）
